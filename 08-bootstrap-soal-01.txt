1. Apa itu CSS Framework?
Jawab: CSS Framework merupakan kumpulan kode CSS yang dapat digunakan untuk mengembangkan website. CSS Framework adalah hasil formula codingan website yang telah siap digunakan. 

2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
Jawab: Tidak

3. Sebutkan beberapa CSS Framework selain Bootstrap
Jawab: Tailwind, materialize, semanticUI, Bulma dan Foundation

4. Bagaimana tata cara penggunaan Bootstrap?
Jawab: menghubungkan halaman ke link CDN yang sudah disediakan oleh bootstrap. Cara penggunaannya dengan memanggil class-class dari bootsrap.
